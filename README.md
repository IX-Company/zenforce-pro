## ✨ Características

- ⏱️ **Pomodoro Timer** con círculo de progreso y 3 fases
- 📋 **Kanban Board** con drag & drop
- 🎨 **6 Temas Premium**: Oscuro, Claro, Neon, Sunset, Forest, Galaxy
- 🌐 **Multi-idioma**: Español / English
- 📊 **Estadísticas** completas (sesiones, minutos, racha)
- 🎯 **Deep Focus Mode** para máxima concentración
- 💾 **Persistencia** automática en localStorage
- 📤 **Exportar/Importar** configuración y tareas
- ⌨️ **Atajos de teclado**: Space, R, S
- 📱 **Responsive** para todos los dispositivos
